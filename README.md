# Программирование — это про тебя

Каждый день ты пользуешься приложениями, сайтами, играми. Всё это создали обычные люди. И ты можешь стать одним из них.

Программирование — это не про сложную математику. Это про **логику** и **творчество**. Ты придумываешь идею — компьютер помогает её реализовать. Как конструктор, только цифровой.

## Как начать

С **MiMoCode** не нужно учить язык с нуля. Ты описываешь задачу на человеческом языке — он пишет код и выполняет команды. Ошибки? Не проблема — он поможет исправить.

```
npm install -g mimocode
mimocode
```

## Что можно создать

- **Сайты** — визитки, портфолио, блоги
- **Приложения** — калькуляторы, планировщики, утилиты
- **Игры** — от простых браузерных до сложных
- **Ботов** — помощников для рутины
