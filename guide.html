Как менять текст

Открой index.html или guide.html в любом текстовом редакторе. Найди нужный текст между HTML-тегами и замени его. Сохрани файл — обнови страницу в браузере.

Текст находится между открывающим и закрывающим тегом. Например, <p>Текст здесь</p>.

Как обновлять сайт

  1. Отредактируй файлы в папке проекта
  2. Открой GitHub Desktop
  3. Напиши сообщение коммита (что изменил)
  4. Нажми Commit to master
  5. Нажми Push origin

Через 1-2 минуты изменения появятся на сайте.

Термины

Git
Программа для сохранения версий кода. Как «Сохранить» в Word, но для всего проекта.
GitHub
Сайт, где хранятся проекты на Git. Доступ из любого компьютера.
Commit
Одно сохранение изменений. Фиксирует текущее состояние файлов.
HTML
Язык разметки — структура и содержание страницы.
CSS
Язык стилей — цвета, расположение, шрифты.
Хостинг
Сервер, где живёт сайт 24/7. Без него сайт виден только тебе.
Домен
Адрес сайта в интернете. Например, shanixxx.github.io.

Как сайт попал в интернет

Используется GitHub Pages — бесплатный хостинг от GitHub.

  1. Создан репозиторий shanixxx.github.io на GitHub
  2. Файлы загружены через GitHub Desktop
  3. В настройках репозитория включён GitHub Pages
  4. Сайт доступен по адресу shanixxx.github.io
