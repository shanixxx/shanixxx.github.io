Как менять текст

Открой index.html или guide.html в текстовом редакторе.

Измени текст между тегами. Сохрани — обнови страницу в браузере.

Как обновлять сайт

1 Отредактируй файлы
2 Открой GitHub Desktop
3 Напиши сообщение коммита
4 Нажми Commit to main
5 Нажми Push origin

Через 1-2 минуты изменения появятся на сайте.

Как разместить сайт в интернете

Хостинг

Это как аренда квартиры для твоего сайта. Сервер работает 24/7 и показывает сайт всем.

Домен

Адрес сайта в интернете. Например, shanixxx.github.io.

GitHub Pages

Бесплатный хостинг. Загружаешь файлы на GitHub — сайт работает.
