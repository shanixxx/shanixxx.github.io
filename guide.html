Как менять текст

Открой index.html или guide.html в любом текстовом редакторе. Замени текст между тегами. Сохрани — обнови страницу в браузере.

Как обновлять сайт

  1. Отредактируй файлы
  2. Открой GitHub Desktop
  3. Напиши сообщение коммита
  4. Нажми Commit to master
  5. Нажми Push origin

Через 1-2 минуты изменения появятся на сайте.

Как сайт попал в интернет

Используется GitHub Pages — бесплатный хостинг от GitHub.

  1. Создан репозиторий shanixxx.github.io
  2. Файлы загружены через GitHub Desktop
  3. В настройках репозитория включён GitHub Pages
  4. Сайт доступен по адресу shanixxx.github.io
