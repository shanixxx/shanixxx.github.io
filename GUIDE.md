## Как это работает

### index.html — скелет сайта
Каждый сайт — это HTML-файл. Вот что внутри:

- `<header>` — верхняя часть (герой-баннер с именем)
- `<section id="about">` — блок "Почему программирование"
- `<section id="projects">` — блок "Чем занимаюсь"
- `<section id="deploy">` — блок "Как разместить сайт в интернете"
- `<footer>` — подвал сайта

**Как менять текст:** открой `index.html` в текстовом редакторе (Notepad++, VS Code) и отредактируй содержимое между тегами. Например, `<h1>[Имя]</h1>` замени на своё имя.

### css/style.css — одежда сайта
Здесь описано, как сайт выглядит:

- Цвета задаются в блоке `:root` (переменные `--bg`, `--text`, `--accent`)
- Адаптивность — сайт подстраивается под размер экрана (секция `@media`)
- Стили для каждой секции привязаны по классам (`.hero`, `.section`, `.feature`)

## Как обновлять сайт

1. Отредактируй файлы в `C:\mimo`
2. Открой GitHub Desktop
3. Увидишь изменения слева — напиши сообщение коммита
4. Нажми **Commit to main**
5. Нажми **Push origin**
6. Через 1-2 минуты изменения появятся на сайте

## Что можно улучшить

- **Заменить `[Имя]** на своё имя в index.html
- **Добавить фотографию** (положи в папку `images/` и добавь `<img>` в HTML)
- **Изменить цвета** в CSS (поменяй значения `--accent` и `--bg`)
- **Добавить новые секции** (скопируй существующую секцию в HTML и измени)
- **Настроить свой домен** (купить домен и привязать к GitHub Pages)
