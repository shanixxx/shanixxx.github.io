Как разместить сайт в интернете

Твой сайт пока живёт только на твоём компьютере. Чтобы другие могли его видеть, нужно загрузить на сервер.

Хостинг

Это как аренда квартиры для твоего сайта. Сервер — это мощный компьютер, который работает 24/7 и показывает твой сайт всем, кто заходит по адресу.

Домен

Это адрес твоего сайта в интернете. Например, moysait.ru или alexey.dev. Без домена люди не смогут найти тебя.

Как загрузить файлы

Самый простой способ — GitHub Pages. Это бесплатно и не требует настройки сервера. Загружаешь файлы на GitHub — и сайт работает.

Чеклист для размещения

  • Создай аккаунт на GitHub.com
  • Установи Git и настрой его (MiMoCode поможет)
  • Создай репозиторий для проекта
  • Загрузи файлы сайта
  • Включи GitHub Pages в настройках
  • Готово! Сайт доступен по адресу

Совет: Когда будешь готов — попроси MiMoCode помочь с размещением. Он подскажет, что делать на каждом шаге.

Как менять текст

Открой index.html в текстовом редакторе (Notepad++, VS Code). Замени [Имя] на своё имя. Сохрани — обнови страницу в браузере.

Как обновлять сайт

1 Отредактируй файлы
2 Открой GitHub Desktop
3 Напиши сообщение коммита
4 Нажми Commit to main
5 Нажми Push origin

Через 1-2 минуты изменения появятся на сайте.
